Anleitung

Zugriff auf die App

Wenn Sie von uns einen Account erhalten haben, können sie unter "Übersicht"oder "Accounteinstellungen" auf den zugangsbeschränkten Bereich zugreifen.

Obwohl Sie von uns bereits einen Account angelegt bekommen haben, können sie bei Bedarf Ihr Passwort ändern. Dazu gehen sie oben auf "Accounteinstellungen" und melden sich, sofern noch nicht geschehen, mit Ihrem aktuellen Zugangsdaten an. Auf der folgenden Seite geben Sie einfach ein neues Passwort ein und klicken auf "Absenden"

Datenerfassung

Übersicht

Der Haupt-Teil dieser Anwendung befindet sich unter "Übersicht". Sie finden dort eine Liste mit allen noch zu bearbeitenden Stolpersteinen.

Unter "Suchen" können Sie nach Einträgen z.B. nach Namen von Opfern suchen. Wenn sie den Kasten neben "Erledigte Einträge ausblenden" anklicken, werden Einträge, die ein:e Admimistrator:in als erledigt markiert hat, ausgeblendet. In der Spalte "Version" sehen sie, wie oft ein Eintrag bereits bearbeitet wurde. In der Spalte "Fertig" sehen sie, ob ein Eintrag als fertig markiert wurde oder nicht. Sie können nur unfertige Einträge bearbeiten.

Eintrag bearbeiten

Sie können einen Eintrag bearbeiten, indem sie auf den Namen eines Opfers klicken. Gehen sie nun auf den Link zum Eintrag auf der alten Website. Füllen sie leere Felder aus und überprüfen sie bereits ausgefüllte Felder. Manche Felder sind mit Anweisungen versehen. Bewegen sie ihre Maus über die Fragezeichen neben den Feldern, um die Anweisung anzuzeigen.

Klicken sie auf "Absenden" um ihren Eintrag zu beenden
